◆ begin()

bool Adafruit_INA3221::begin ( uint8_t  i2c_addr = INA3221_DEFAULT_ADDRESS,
TwoWire *  theWire = &Wire 
)

Initialize the INA3221 device with default settings.

After initialization, the function performs the following configurations:

  • Enables all three channels.
  • Sets conversion time for both bus and shunt to 1ms.
  • Sets averaging mode to 1x.
  • Sets mode to continuous bus and shunt reading.
Parameters
i2c_addrThe I2C address of the INA3221 device. Default is INA3221_DEFAULT_ADDRESS.
theWireThe I2C bus to use. Default is &Wire.
Returns
bool True if initialization and configuration were successful, false otherwise.

◆ setPowerValidLimits()

bool Adafruit_INA3221::setPowerValidLimits ( float  vlimitlow,
float  vlimithigh 
)

Set the Power-Valid upper and lower limits.

This function writes the specified upper and lower voltage limits to the INA3221's Power-Valid registers. The voltage values are converted into a 16-bit register value (each LSB represents 8 mV).

Parameters
vlimitlowThe lower voltage limit in volts.
vlimithighThe upper voltage limit in volts.
Returns
true if both writes were successful, false otherwise.
